# Building

1. Install BepInEx
2. Install Jotunn
3. Update src/build/Environment.props
	- `<VALHEIM_INSTALL>`
	- `<BEPINEX_PATH>`
4. Build Solution
	- The mod will be deployed to the bepinex plugin based on Environment.props
5. Start will run Valheim using the paths given in Environment.props
